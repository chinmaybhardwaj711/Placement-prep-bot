# 🎯 Placement Prep Bot

An AI-powered RAG chatbot built using:

- LangChain
- FAISS
- Hugging Face Embeddings
- Groq Llama 3.1
- Streamlit
- EasyOCR

## Features

- 📄 Multi-PDF upload
- 🧠 Conversational memory
- 🔍 Source citations
- 📊 Retrieval scores
- ✍️ OCR support for handwritten PDFs
- 🎓 Subject-specific interview mode
